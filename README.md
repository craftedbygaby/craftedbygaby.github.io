# Gabriela Cascione — Data Analyst Portfolio

A personal portfolio website built with HTML and CSS, designed to showcase my data analytics projects, professional experience, and skills.

---

## What this project does

This is a static portfolio website containing:
- A project showcase with data analyses (SQL, Python, Tableau)
- A skills and languages overview
- A professional experience timeline
- Testimonials from colleagues and managers
- A contact section with links to LinkedIn, GitHub, and Tableau Public
- An Impressum, as required by German law for websites operated from Germany

---

## Why this project is useful

This portfolio serves as a living CV — a place where recruiters, hiring managers, and collaborators can get a full picture of my work beyond what fits on a one-page document. Each project links directly to GitHub repositories or Tableau Public dashboards, so the work speaks for itself.

---

## How to get started

The site is live at:
**[https://craftedbygaby.github.io](https://craftedbygaby.github.io)**

To run it locally:
1. Clone or download this repository
2. Make sure `index.html` and `style.css` are in the same folder
3. Open `index.html` in any browser — no server or installation needed

To download my CV directly, use the button on the website or find `Gabriela_Cascione_CV.pdf` in this repository.
